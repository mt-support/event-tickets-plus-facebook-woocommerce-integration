Integrations Event Tickets Plus with Facebook for WooCommerce (allowing ticket products to be published in the linked
Facebook shop).

== Description ==

This extension was written to enable easier integration between
[Event Tickets Plus](https://theeventscalendar.com/product/wordpress-event-tickets-plus/) and
[Facebook for WooCommerce](https://woocommerce.com/products/facebook/). It was written to work with the following
plugin versions and your mileage may vary if you are using anything else:

* Event Tickets Plus *4.6.2*
* WooCommerce *3.3.1*
* Facebook for WooCommerce *1.7.5*

If you experience any problems or wish to share any feedback, please feel to
[reach out and let us know](https://theeventscalendar.com/support/forums/forum/others/extensions/) - we'll be happy to
review your notes, but do understand that ongoing support of free, optional extensions like this one is not guaranteed.
